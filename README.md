# PokeDex

Um site moderno e responsivo de Pokémon que utiliza a PokeAPI para exibir informações detalhadas sobre seus Pokémons favoritos.

## Funcionalidades

- **Busca Dinâmica**: Pesquise qualquer Pokémon pelo nome ou número da Pokédex.
- **Interface Premium**: Design moderno com efeito "glassmorphism" (vidro fosco) e animações suaves.
- **Dados em Tempo Real**: Exibe imagem oficial, tipos e estatísticas base (HP, Ataque, Defesa, Velocidade).
- **Responsivo**: Funciona perfeitamente em computadores e dispositivos móveis.
- **Tratamento de Erros**: Mensagens amigáveis quando um Pokémon não é encontrado.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização avançada com variáveis, flexbox, grid e animações.
- **JavaScript (Vanilla)**: Lógica para consumo da API e manipulação do DOM.
- **PokeAPI**: Fonte de dados dos Pokémons.

## Como Usar

1. Clone este repositório ou baixe os arquivos.
2. Abra a pasta do projeto.
3. Dê um clique duplo no arquivo `index.html` para abrir no seu navegador.
4. Digite o nome de um Pokémon (ex: "pikachu", "charizard") e pressione Enter ou clique no botão de busca.

## Créditos

Desenvolvido como um projeto de demonstração focado em UI/UX moderna e consumo de APIs.
